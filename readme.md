# PDF Splitter by names.txt

Веб-приложение на Streamlit для разделения одного PDF на несколько PDF-файлов по списку из `names.txt`.

## Что умеет

- загружает один общий PDF
- загружает `names.txt`
- для каждого будущего PDF берет имя файла и количество страниц
- формирует ZIP-архив с готовыми PDF

## Формат names.txt

Каждая строка:

```txt
имя_файла.pdf,количество_страниц
